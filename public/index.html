Tic Tac Toe vs JavaScript

Click Here!

Tic Tac Toe vs JavaScript is a fun game created with the intention of playing against an "AI". Javascript was the main language used. To generate the "AI's" moves the Math.random and Math.floor functions were used to generate random indexes on the grid. To ensure JavaScript can only play unclicked boxes if/else statements were used. The grid boxes were created by using nested for-loops and an event listener was used to track the users moves. The goal was to create a simple Tic Tac Toe game that keeps both scores and notifies the user if they've won, lost or if it's a draw.

Cocktails & Quotes

Click Here!

Cocktails & Quotes was a front-end project where 2 API's were used to create a fun responsive app that allows you to make a drink and read Ron Swanson Quotes with your friends. AJAX/fetch was used to fetch the drinks/ingredients when the user searched for a specific drink name like 'Margarita'. Error handling was also incorporated for cases of unknown or misspelled drinks. I was in charge of connecting both API's, printing out all drink options with their designated ingredients and measurements and website responsiveness/design.

Foodie ATL

Still in Production!

Foodie ATL is a backend project that incorporates a database and contains 3 of the CRUD operations. We created our own database with restaurants we'd like to or have already tried in the metro atlanta area. We wanted to create an app similar to tinder but with food for those days you can't decide where you'd like to eat. The page allows you to swipe left or right on a restuarant and if you're not sure you can click on the display image and get more info about the restaurant like a photo of the menu, the price point and the hours of operations. When you find a place you'd like to go to you'd like it and that restaurant is added to your favorites list.
